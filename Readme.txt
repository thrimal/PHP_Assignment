# Project Name

## Description
This project is a PHP web application developed using XAMPP server and PHPMyAdmin. It serves as a template for starting PHP-based projects with a MySQL database backend.

## Requirements
- [XAMPP](https://www.apachefriends.org/index.html): A free and open-source cross-platform web server solution stack package developed by Apache Friends.
- [PHPMyAdmin](https://www.phpmyadmin.net/): A free software tool written in PHP intended to handle the administration of MySQL over the web.

## Setup Instructions
1. **Install XAMPP:**
   - Download and install XAMPP from the [official website](https://www.apachefriends.org/index.html) according to your operating system.
   - Follow the installation wizard instructions.
   
2. **Start XAMPP:**
   - Open XAMPP Control Panel.
   - Start the Apache and MySQL services by clicking on their respective "Start" buttons.

3. **Access PHPMyAdmin:**
   - Open a web browser and navigate to `http://localhost/phpmyadmin/`.
   - Log in using the default credentials (if you haven't changed them during installation).
   - doesn't require mysql password. should be empty this field.

4. **Create a Database:**
   - Click on the "Databases" tab.
   - Enter a name for your database and click on "Create".
   - import the given database file.

5. **Run Project**
   - Before you run or start writing any program in PHP, you should start Apache and MYSQL using XAMPP control panel. 
   - Then copy that project folder to C:/XAMPP/htdocs.
   - Open the browser and type http://localhost/assignment.
   - Thats all.
